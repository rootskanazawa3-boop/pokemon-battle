# GitHubアカウント作成と公開の手順

## ステップ1: GitHubアカウントを作成

1. [GitHub.com](https://github.com) にアクセス
2. 右上の「Sign up」ボタンをクリック
3. 以下の情報を入力：
   - **Username（ユーザー名）**: 他の人が使っていないユーザー名（例: `yourname-pokemon`）
   - **Email address（メールアドレス）**: 使用可能なメールアドレス
   - **Password（パスワード）**: 安全なパスワード
4. メールアドレス確認のコードを入力
5. アカウント作成完了

## ステップ2: リポジトリを作成

1. GitHubにログイン後、右上の「+」ボタン → 「New repository」をクリック
2. リポジトリ名を入力（例: `pokemon-battle`）
3. 「Public」を選択（無料で公開できます）
4. ⚠️ **「Initialize this repository with a README」はチェックしない**
5. 「Create repository」をクリック
6. リポジトリが作成されると、URLが表示されます
   - 例: `https://github.com/YOUR_USERNAME/pokemon-battle.git`

## ステップ3: リポジトリURLを教えてください

リポジトリ作成後、以下のいずれかを教えてください：
- リポジトリのURL（`https://github.com/...`）
- または、GitHubのユーザー名とリポジトリ名

その後、こちらで公開まで進めます！

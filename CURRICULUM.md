## 📅 詳細な授業プラン

### 第1週：環境構築と基礎理解

#### **第1回：環境構築（2時間）**
**目標**: 開発環境を整える

**内容**:
1. Node.jsのインストール確認
2. プロジェクトの作成（Vite + React + TypeScript）
   ```bash
   npm create vite@latest pokemon-battle -- --template react-ts
   cd pokemon-battle
   npm install
   ```
3. Tailwind CSSの設定
4. 開発サーバーの起動（`npm run dev`）
5. ブラウザで確認

**課題**: ローカルで開発サーバーを起動できるようになる

---

#### **第2回：React基礎とコンポーネント設計（2時間）**
**目標**: Reactの基本を理解し、コンポーネント設計を学ぶ

**内容**:
1. Reactの基本概念（コンポーネント、props、state）
2. TypeScriptの型定義
3. コンポーネント設計の考え方
   - どこまでを1つのコンポーネントにするか
   - 親子コンポーネントのデータの受け渡し
4. プロジェクト構造の理解

**実習**: 簡単なカードコンポーネントを作成

---

#### **第3回：データ構造の設計（2時間）**
**目標**: ゲームのデータ構造を設計する

**内容**:
1. モンスター（Bokemon）の型定義
   - id, name, hp, attack, defense, moves など
2. 技（Move）の型定義
3. バトル状態（BattleState）の型定義
4. 初期データの作成

**実習**: 型定義ファイルを作成し、ダミーデータでモンスターを表示

**コード例**:
```typescript
interface Bokemon {
  id: number;
  name: string;
  hp: number;
  maxHp: number;
  attack: number;
  defense: number;
  type: string;
  moves: string[];
}
```

#### **第5回：状態管理（useState）（2時間）**
**目標**: Reactの状態管理を理解する

**内容**:
1. useStateフックの基本
2. 複数の状態の管理方法
3. 状態の更新タイミング
4. プレイヤー・敵のモンスター状態の管理

**実習**: モンスター選択画面を作成し、選択したモンスターを表示

**コード例**:
```typescript
const [playerTeam, setPlayerTeam] = useState<Bokemon[]>([]);
const [playerBokemon, setPlayerBokemon] = useState<Bokemon | null>(null);
```

#### **第6回：バトルロジックの実装（2時間）**
**目標**: バトルの核となるロジックを実装する

**内容**:
1. ダメージ計算のアルゴリズム
2. HP更新の処理
3. ターン制の実装
4. 勝敗判定のロジック

**実習**: シンプルな1ターンのバトルを実装

**コード例**:
```typescript
const calcDamage = (attacker: Bokemon, defender: Bokemon, move: Move): number => {
  const base = ((attacker.attack / defender.defense) * move.power) + 2;
  const random = 0.85 + Math.random() * 0.15;
  return Math.max(1, Math.floor(base * random));
};
```
